## VIM cheat sheet

### :Commands

#### Buffers
* :bufdo e! - re-read all buffers from disk (e.g. after "git pull")

#### Plugins
* :CtrlPClearAllCaches - clear cache of CtrlP plugin, if we have new files to search

* * *

### Custom Key Maps

* nmap ,d :b#<bar>bd#<CR> - switch to previous buffer && close current
